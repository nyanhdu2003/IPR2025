# Run commands

## Run application with dotnet cli

1. Restore the solution

    ```bash
    dotnet restore
    ```

2. Build the solution

    ```bash
    dotnet build
    ```

3. Run QRPackingApp.API

    ```bash
    dotnet run --project QRPackingApp.WebAPI
    ```

4. Dotnet Watch
    
    ```bash
    dotnet watch --project QRPackingApp.WebAPI
    ```
